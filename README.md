# Sistema de Gerenciamento de Tarefas – Projeto Ágil

## Objetivo
Desenvolver um sistema web simples para gerenciamento de tarefas, pela empresa TechFlow Solutions aplicando conceitos de Engenharia de Software, com metodologias ágeis e controle de qualidade.

## Escopo Inicial
O sistema permite:
- Criar tarefas
- Listar tarefas
- Editar tarefas
- Excluir tarefas

## Tecnologias
- Python
- Flask
- Pytest
- GitHub Actions

## Estrutura do Projeto
/src – código fonte  
/tests – testes automatizados  

## Metodologia Utilizada
Foi adotado o Kanban, utilizando o GitHub Projects para organizar as atividades nas colunas To Do, In Progress e Done.

## Mudança de Escopo
Durante o desenvolvimento, foi adicionada a funcionalidade de prioridade de tarefas, permitindo classificar cada tarefa como baixa, média ou alta prioridade.

## Controle de Qualidade
Foi configurado um pipeline de integração contínua com GitHub Actions para executar automaticamente os testes a cada commit.
